
# Evohome2MQTT Add-on for Home Assistant

Ce module complémentaire connecte votre système Honeywell Evohome à MQTT pour intégration dans Home Assistant.

## Options
- `mqtt_host`: Adresse du broker MQTT
- `mqtt_port`: Port du broker MQTT
- `mqtt_user`: Utilisateur MQTT
- `mqtt_password`: Mot de passe MQTT
- `evohome_user`: Email Evohome
- `evohome_password`: Mot de passe Evohome
- `polling_interval`: Intervalle en secondes (par défaut 30)
